# Ejercicio fechas

* Crea un POJO llamado User con los siguiente campos:
  * nif
  * name
  * surnam
  * birthDate: LocalDate
* Crea un Reader para este POJO: debes pedir una fecha en formato String y convertirla en LocalDate
* Crea un Main que
  * Crea el reader
  * Pide dos usuarios
  * Comprueba cual de ellos es el mayor de los dos y muestra sus datos en pantalla
  * Luego comprueba cuales de ellos son mayores de edad y los muestra si son mayores de 18
